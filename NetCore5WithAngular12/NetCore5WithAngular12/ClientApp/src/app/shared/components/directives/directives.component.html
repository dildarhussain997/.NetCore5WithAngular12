*ngIf can be used to hide or show and element but hidden property can also be used for that

diffrence between *ngIf and hidden property is that *ngIf does not add element to the DOM while hidden property add element to DOM but its hidden from user view but still present in DOM can be seen from inspect element

best practice: if elemnt is small then use hidden property as it does not takes too much resource and processing power but if element is large then use *ngIf so that elemnt is never added to DOM and save memory and processing

but if an element is large and is needed again again and takes large processing time then its better to hide it with hidden property so that we dont need to get data for that element again
